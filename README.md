# Flask Student API

This is a simple backend project built using **Python and Flask**.
The API allows basic management of student data such as creating, viewing, updating and deleting student records.

## What this project does

* Add a new student
* View all students
* View a single student by ID
* Update student information
* Delete a student record

This project was created to practice **Flask and REST API development**.

## Tech Used

* Python
* Flask
* SQLite / SQL
* REST API

## API Routes

GET /students – get all students
GET /students/<id> – get student by id
POST /students – create new student
PUT /students/<id> – update student
DELETE /students/<id> – delete student

## How to run the project

Clone the repository
git clone https://github.com/originalanimemasters-prog/flask-student-api.git
Go to the project folder
cd flask-student-api
Install dependencies
pip install -r requirements.txt
Run the app
python app.py

The server will start on
http://127.0.0.1:5000/
